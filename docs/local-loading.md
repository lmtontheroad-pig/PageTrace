# Local Loading

1. Open Chrome and go to `chrome://extensions/`.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select the repository root directory that contains `manifest.json`.
5. Open a matched page under `https://www.pornhub.com/`.
6. Use the extension popup to check hit counts, pause, enable, or rescan.
7. Open the extension options page to add, delete, or edit keyword colors.

After changing source files, reload the extension from `chrome://extensions/`, then refresh the matched page.
